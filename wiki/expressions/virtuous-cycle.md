## Meaning
A self-reinforcing positive feedback loop where each improvement leads to further improvements. Korean: 선순환. Opposite: vicious cycle (악순환).

## Examples
- "Human corrections feed back into the model — creating a **virtuous cycle** of improvement."
- "More data → better model → more users → more data: that's the **virtuous cycle**."
- "Amazon calls this a **flywheel** — same concept, different metaphor."

## Similar Expressions
- flywheel — Amazon/Bezos framing; same concept (self-reinforcing momentum)
- feedback loop — more neutral; can be positive or negative
- snowball effect — informal; positive momentum building

## Notes
- Pronunciation: VER-choo-us (not "vicvious" or "victorious")
- Common in business/product interviews — good word to use naturally
- Pair it with "flywheel" for extra credibility: "a virtuous cycle — what Amazon calls a flywheel"
