# Question Formation Errors

## Pattern
Mixing auxiliary verbs in questions. Common error: using "is" + action verb instead of "does" + base form.

|------|-------------|-------------|--------|

## Review Notes
- **"How much is it?"** — using "be" verb, no action verb needed
- **"How much does it cost?"** — using "do/does" with action verb "cost"
- **Never combine both**: ❌ "How much is it cost?"
- Rule of thumb: if there's an action verb, use do/does/did for the question
