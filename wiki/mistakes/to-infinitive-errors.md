# To-Infinitive Errors

## Pattern
Using the wrong verb form after "to". The correct structure is **to + base form** (e.g., "to eat", "to go", "to order"), NOT "to + -ing".

## Review Notes
- **to + base form**: I want **to order**. / She needs **to leave**. / They decided **to stay**.
- **Common verbs followed by to-infinitive**: want, need, decide, plan, hope, expect, agree, offer, promise, refuse
- Don't confuse with gerund (-ing): some verbs take -ing (enjoy, avoid, finish), some take to-infinitive (want, need, decide)
