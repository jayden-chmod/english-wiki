# Noun-Verb Confusion

## Pattern
Using a noun form where a verb is needed (or vice versa). Common with word pairs like improve/improvement, develop/development, achieve/achievement.

## Review Notes
- "improvement" = noun (a thing): "This is an improvement."
- "improve" = verb (an action): "I want to improve."
- Quick check: if it follows "can / will / want to / try to" → you need the verb (base form)
- ❌ "I want to improvement" → ✓ "I want to improve"
- Common pairs: improve/improvement, develop/development, achieve/achievement, organize/organization
